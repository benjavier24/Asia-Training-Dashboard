# Asia Training Dashboard - AI Powered

## Quick Start

1. Open a terminal/command prompt
2. Navigate to this folder:
   ```
   cd "c:**\Users\BenjJavier\OneDrive - bolttech\Documents\Copilot\Created\training_dashboard**"
   ```
3. Run the dashboard:
   ```
   py -m streamlit run app.py
   ```
4. Your browser will open automatically at http://localhost:8501

## Features

- **AI-Powered Insights**: Auto-generated analysis of your training data
- **Interactive Filters**: Filter by Country, Account, Trainer
- **6 Key Metrics**: Participants %, Trainings, Pass Rate, Attach Before/After, Improvement
- **3 Input Methods**: Paste from Excel, Upload file, or Demo data
- **Smart Recommendations**: AI suggests actions based on performance gaps
- **Export**: Download filtered data as CSV or Excel

## How Trainers Use It

1. Open dashboard (bookmark the URL after first launch)
2. Choose "Paste Data" in sidebar
3. Copy data from Excel (Ctrl+A, Ctrl+C)
4. Paste into the text area
5. Dashboard updates automatically
